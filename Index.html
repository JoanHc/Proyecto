Diseño Web

El diseño web es un área enfocada en el desarrollo de interfaces digitales, como el diseño de sitios y aplicaciones para web. Para ello, los diseñadores web crean las páginas utilizando lenguajes de marcado como HTML.

E-Commerce

El e-commerce o comercio electrónico consiste en la distribución, venta, compra, marketing y suministro de información de productos o servicios a través de Internet.

Aplicaciones móviles

Una aplicación móvil, también llamada app móvil, es un tipo de aplicación diseñada para ejecutarse en un dispositivo móvil, que puede ser un teléfono inteligente o una tableta. ... Por ejemplo, puede ser un juego, una calculadora o un navegador web móvil.
